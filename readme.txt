	本项目实现了登录验证、分页、页面容量、数据删除、数据更新功能。

	1.登录成功则直接跳转index.html页面，若登录失败则跳回登录页面重新登录。
	2.分页功能按钮和页面容量功能已全部实现。
	2.对于删除和更新按钮在页面右边的功能按钮处。
	3.删除功能只能实现删除后来添加的数据。对数据库中原有的有主外键关联的数据不能删除。
	4.更新功能点击后会弹出新页面进行更新操作（原始数据会默认显示）。
